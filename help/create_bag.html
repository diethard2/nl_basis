Create BAG database

First select the output folder where you want to create the bag database. If you select the folder C:\data\bag, the database C:\data\bag\bag.sqlite will be created.

Select the input folder, this folder will contain the two downloaded files that are used to read the information to create the BAG database.

The file that is downloaded is a zip-file holding many XML files. Never extract this zip file! It is a lot of data (about 60 GB) zipped it is only 1.4 GB.

Processing will take a long time. It is a lot of data to process and it can take several hours. The better the hardware, the faster the processing.
