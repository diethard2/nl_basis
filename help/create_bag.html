Create BAG database

Description

It is used to process a big zip file that contains information off all buildings of the Netherlands including addresses.
These xml-files can not be read into QGIS without processing them first.

Currently this plugin is developed, tested and working on Windows OS only.
Soon it will be supported on Linux and OSX OS as well.

First download the following two files using the link and place only those files in the folder that will be referred to as the input folder:

  1. inspireadressen.zip
    http://geodata.nationaalgeoregister.nl/inspireadressen/extract/inspireadressen.zip
  2. Tabel33 Gemeententabel (gesorteerd op omschrijving).csv
    http://www.rijksdienstvooridentiteitsgegevens.nl/dsresource?objectid=16993&type=org

The file inspireadressen.zip that is downloaded is a zip-file holding many XML files. Never extract this zip file! It is a lot of data (about 60 GB) zipped it is only 1.4 GB.

Parameters

Input folder

This folder should contain the two downloaded files that are used to read the information to create the BAG database.

Output folder

The output folder is where you want to create the bag database.
If you select the folder C:\data\bag, the database C:\data\bag\bag.sqlite will be created.

Additional information

Processing will take a long time. It is a lot of data to process and it can take several hours. The better the hardware, the faster the processing.

There are minimal requirement for the hardware.
Make sure you have at least:

  • 20 GB diskspace available
  • 4 GB of internal memory

For testing purposes there is a smaller inspireadressen.zip available that can be obtained using the next link referring to a public google drive folder:
